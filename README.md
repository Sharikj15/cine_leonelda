# Cine Leonelda

## ⚠️ Disclaimer - Propósito Educativo

**IMPORTANTE:** Este proyecto es completamente educativo. **Ningún valor, dato, precio, información o contenido presentado en esta aplicación es real**. Todo el contenido, incluyendo:

- Nombres de películas y detalles
- Precios y tarifas
- Información de usuarios
- Datos de transacciones
- Cualquier otra información mostrada

...es **ficticio y generado únicamente con fines educativos** para demostrar conceptos de desarrollo web.

---

## Descripción

Cine Leonelda es un proyecto educativo que simula una plataforma de compra de entradas de cine, desarrollado para aprender y practicar conceptos de:

- Desarrollo web frontend
- Gestión de rutas y componentes
- Autenticación y validación de formularios
- Gestión de datos y API
- Diseño responsivo y UX

## Características

- Sistema de login educativo
- Catálogo de películas (ficticias)
- Carrito de compras (simulado)
- Gestión de usuarios
- Interfaz responsiva

## Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)
- JSON (para datos locales)
